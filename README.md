# Procedural-Ocean-URP

**This is a WIP**

Procedural ocean landscape generator for Unity.
Currently includes:
- Simple terrain generation.
- Ocean simulation:
  - Wave simulation
  - Underwater sun shafts
  - Caustics
- Plant Generation:
  - Corals with space colonization algorithm.
  - Corals with reaction-diffusion.
  - Giant kelp
  - Seaweed
